Java基础(一)

一、注释

1.什么是注释

源文件中用于解释说明程序的文字就是注释

2.Java中注释的类型

  • 单行注释
  • 多行注释
  • 文档注释（java特有）

3.注释的作用

  • 提高程序的可阅读性
  • 是调试程序的重要方法

注意：

  • 单行注释与多行注释不参与程序的编译运行
  • 多行注释不能嵌套使用
  • 文档注释内容可以被JDK提供的工具javadoc所解析，生成一套以网页文件形式体现该文件的说明文档

二、关键字（keyword）

  • 定义：被Java语言赋予了特殊含义，用做专门用途的字符串（或单词）
  • 特点：全部关键字都是小写字母。
  • 关键字比较多，不需要死记硬背，学到哪里记到哪里即可。
  • 官方地址： https://docs.oracle.com/javase/tutorial/java/nutsandbolts/_keywords.html

说明：

  1. 关键字一共50个，其中const和goto是保留字(reserved word)。
  2. true，false，null不在其中，它们看起来像关键字，其实是字面量，表示特殊的布尔值和空值。

用于定义数据类型的关键字：

class interface enum byte short
int long float double char
boolean void

用于定义流程控制的关键字:

if else switch case default
while do for break continue
return

用于定义访问权限修饰符的关键字：

private protected public

用于定义类、函数、变量修饰符的关键字：

abstract final static synchronized

用于定义类与类之间关系的关键字：

extends implements

用于定义建立实例及引用实例，判断实例的关键字：

new this super instanceof

用于异常处理的关键字：

try catch finally throw throws

用于包的关键字：

package import

其他修饰符关键字：

native strictfp transien volatile assert
const goto

用于定义数据类型的字面值：

true false null

三、标识符

Java中变量、方法、类等要素命名时使用的字符序列，称为标识符。

凡是自己可以起名字的地方都叫标识符。

  • 标识符命名规则（必须遵守的硬性规定）：

    • 由26个英文字母大小写，0-9 ，_或 $ 组成

    • 数字不可以开头。

    • 不可以使用关键字和保留字，但能包含关键字和保留字。

    • Java中严格区分大小写，长度无限制。

    • 标识符不能包含空格。

  • 标识符命名规范（建议遵守的约定俗成）：

    • 包名：多单词组成时所有字母都小写：xxxyyyzzz。
      例如：java.lang、com.atguigu.bean

    • 类名、接口名：多单词组成时，所有单词的首字母大写：XxxYyyZzz
      例如：HelloWorld，String，System等

    • 变量名、方法名：多单词组成时，第一个单词首字母小写，第二个单词开始每个单词首字母大写：xxxYyyZzz
      例如：age,name,bookName,main,binarySearch,getName

    • 常量名：所有字母都大写。多单词时每个单词用下划线连接：XXX_YYY_ZZZ
      例如：MAX_VALUE,PI,DEFAULT_CAPACITY

    • 命名要尽量见名知意

四、Java中变量的数据类型

Java语言是强类型语言，对于每一种数据都定义了明确的具体数据类型，在内存总分配了不同大小的内存空间

1.基本数据类型

整数类型 所占空间 表示范围
byte 1个字节 -2^7 ~ 2^7-1
short 2个字节 -2^15 ~ 2^15-1
int 4个字节 -2^31 ~ 2^31-1
long 8个字节 -2^63 ~ 2^63-1
小数类型 所占空间
float 4个字节
double 8个字节
  • 字符类型

​ char 所占空间大小根据实际编码来决定

  • 布尔类型

​ boolean 占1个字节

2.引用数据类型

略

二、变量定义的语句格式

注意事项：
1、在同一作用域（理解为一个大括号内部）下，变量名不能重复
2、整数默认是int类型，要想定义一个long类型的变量，需要在值的后面加上L或者l，推荐L
3、小数默认是double数据类型
4、要想定义一个float类型的变量,需要在值的后面加上F或者f，推荐F
5、遍历必须要有初始化值,否则无法使用

三、数据类型转换

数据类型之间的转换：

  • 自动数据类型转换

    byte,short,char–>int-long-float-double

  • 强制类型转换

    语句定义格式：目标数据类型 变量名 = (目标数据类型)(表达式或者结果)

    注意：强制数据类型转换，有可能会导致精度损失，不建议基本数据类型之间使用强制类型转换

注意事项：

  • 变量之间参与运算，会先自动提升数据类型
  • 常量之间参与运算，会先计算出结果，再判断是否在目标数据类型范围内

题1:

哪句是编译失败的呢？为什么呢？

语句b=b1+b2;是编译失败的，Java中默认的整数类型是int,byte类型的b1和b2在运算中会进行自动数据类型转换成int型，无法被byte类型的变量b接收

结论：

五、运算符

1.基本定义

  • 运算

​ 对常量和变量进行操作的过程称为运算。

  • 运算符

​ 对常量和变量进行操作的符号称为运算符

  • 操作数

​ 参与运算的数据称为操作数

  • 表达式

​ 用运算符把常量或者变量连接起来符号java语法的式子就可以称为表达式。不同运算符连接的式子体现的是不同类型的表达式

举例：

int a = 3 + 4;

这是做了一个加法运算

+就是运算符，是算术运算符，我们还有其他很多的运算符

3，4就是参与运算的操作数据

3 + 4整体其实就是一个算数表达式

2.运算符

2.1算术运算符

运算符 运算 范例 结果
+ 正号 +3 3
- 负号 b = 4 ; - b - 4
+ 加号 5 + 5 10
- 减号 6 - 4 2
* 乘号 3 * 4 12
/ 除号 6 / 3 2
% 取模(取余) 5 / 5 0
++ 自增（前） a = 2 ; b = ++ a ; a = 3 ; b = 3
++ 自增（后） a = 2 ; b = a ++ ; a = 3 ; b = 2
– 自减（前） a = 2 ; b = – a ; a = 1 ; b = 1
– 自减（后） a = 2 ; b = a – ; a = 1 ; b = 2
+ 字符串相加 “He” + “llo” “Hello”

注意：

  • 表达式最终结果的数据类型，由参与运算中最大的数据类型决定

    byte–short–char——> int–long–float–double

  • 自增(或自减)运算在前时，先自增(或自减),然后在参与运算

    自增(或自减)运算在后时，先参与运算，然后在自增(或自减)

2.2赋值运算符

运算符 范例 结果
= a1 = 4 a1 = 4
+= a2 = 4 ; a2 += 2; a2 = 6
-= a3 = 4 ; a3 -= 2; a3 = 2
*= a4 = 4 ; a4 *= 2; a4 = 8
/= a5 = 4 ; a5 /= 2; a5 = 2
%= a6 = 4 ; a6 %= 2; a6 = 0

注意：

s += 1 ;并不等效于 s = s + 1 ;

2.3关系运算符(比较运算符)

运算符 运算 范例 结果
== 等于 4==3 false
!= 不等于 4!=3 true
< 小于 4<3 false
> 大于 4>3 true
<= 小于等于 4<=3 false
>= 大于等于 4>=3 true
instanceof 检查是否是类的对象 “Hello” instanceof String true

注意：

  • 比较运算符的结果都是boolean型，也就是要么是true，要么是false
  • 比较运算符“==”不能误写成“=”

2.4逻辑运算符

运算符 运算 范例 结果
&(有false则false) 与(AND) false&true false
|(有true则true) 或(OR) false|true true
^(相同则false，不同则true) 异或(XOR) true^false true
!(将true变为false，将false变为true) 非(Not) !true false
&& 短路(AND) false&&true false
|| 短路(OR) false||true true

注意：

  • Java中不允许连续比较，如：3<x<6，应当使用逻辑运算符用于连接布尔型表达式，如：x>3 & x<6
  • “&”和“&&”的区别：
    • “&” 左边无论真假，右边都进行运算；
    • “&&” 如果左边为真，右边参与运算，如果左边为假，那么右边不参与运算。
  • “|”和“||”的区别：
    • “|” 左边无论真假，右边都进行运算
    • “||” 左边为真，右边不参与运算。
  • 异或( ^ )与或( | )的不同之处是：当左右都为true时，结果为false。

2.5位运算符

运算符 运算 范例 结果
&(有0则0) 与运算 3&4 0
|(有1则1) 或运算 3|4 7
^(相同则0，不同则1) 异或运算 3^4 7
~(0变成1,1变成0) 反码 ~4 -5
<< 左移 3<<2 12
>> 右移 24>>2 6
>>> 无符号右移 -24>>2 1073741818

注意：

  • “>>” 补码向右移若干位,多余的舍去,不足的根据最高位补足,最高位为1则补1,否则补0,右移n位等效于除以2^n
  • “<<” 补码向左移若干位,多余的舍去,不足的用0补齐,左移n位等效于乘上2^n
  • “>>>”补码向右移若干位,多余的舍去,不足的用0补齐

运算符的优先级（从高到低）
