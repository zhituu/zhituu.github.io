python基础语法

一、输入和输出

1.输入

  • input()

输入的所有内容都为字符串

  • 特殊用法：用作停顿

2.输出

  • print()

print()源码 ；Ctrl + 鼠标左键选中函数，查看源码

  • sep参数 逗号拼接值的时候，默认是以空格拼接
  • end 指定输出结尾的方式，默认是\n换行

二、变量

  • python提供了一种类似于数学里未知数的表达，将一个值赋值给一个未知数，这个未知数在python里叫做变量

  • 给变量取名遵循的原则

    • 由英文大小写组成，数字，下划线组成
    • 不能以数字开头
    • 不能是python中的关键字

  • 给变量起名字的规范

    • 要见名知意 name
    • 起名字的格式遵循驼峰式或者下划线式

三、基本数据类型

1.字符串 str

  • 单引号或双引号将若干个字符括起来的字符序列叫做字符串
  • 使用三个单引号或三个双引号括起来的字符序列，叫做字符串

（可以多行编写）

2.整数 int

3.小数 float

4.布尔 bool

四、类型转换

代码出错有可能是数据类型错误，需要转换

1.转整型

  • int()

字符串中符合数值的样式才可以转数值，否则报错

2.转字符串

  • str()

使用python内置的type() 函数查看元素数据类型

3.转布尔

  • bool()

整型值为0，小数值为0，字符串为空转布尔都为False,其余都为true

五、注释

用于解释说明的文字，不参与代码解释运行

  • 帮助我们理解代码逻辑
  • 帮助我们排错
  • 单行注释 “#加空格 跟要注释的文字”
  • 多行注释 使用三个三引号或三个双引号括起来

不用变量接收的字符串可以当做注释使用

  • 快捷键：ctrl + / 选中要注释的内容，再按一次取消注释

六、pass
